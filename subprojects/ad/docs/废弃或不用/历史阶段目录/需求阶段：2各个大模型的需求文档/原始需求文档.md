# 小乔智投Agent-产品需求文档 (PRD)

## 1. 产品愿景与目标 (Vision & Objectives)

### 1.1 产品定位
小乔Agent精通广告全链路数据服务，包括数据分析，归因回传，问题排查等。

### 1.2 核心目标
*   **优化师赋能**: 让投放快速获取多维报表数据，轻松掌握归因回传逻辑，高效完成排查、诊断、预测等日常分析活动。
*   **产研增效**: 极大的减少低效产出，专注于AI能力输出。
*   **AI-First 体验**: 以 Agent 矩阵支撑深度操作，交互式agent设计范式便于AI拓展。

---

## 2. 核心功能模块 

### 2.1 智能对话中枢  - 核心入口
*   **交互模式**: 全屏对话流，支持自然语言指令调度所有专项 Agent。

### 2.2 问答 Agent -以专业解答为主
*   **知识问答**: 基于媒体文档（巨量/腾讯/快手）与行业知识图谱，回答优化师关于媒体对接、数据gap疑问。
*   **指标解释**: 标准化定义 ROI, 次留率, LTV 等原子指标及计算过程。
*   **需求收集**: 引导用户提交对接需求（开发者地址、Key、账号），自动标准化并写入需求池。

### 2.3 监控 Agent -以数据可视化为主
*   **链路观测**: 轻松掌握广告数据处理链路、归因与回推逻辑，快速排除系统影响。
*   **风险熔断**: 实时监控系统健康状态，采集延迟、回推延迟、上报延迟、异常流量，支持 AI 自动预警与策略熔断。
*   **数据比对**: 媒体数据与平台数据的自动比对，发现异常波动。

### 2.4 诊断 Agent -以问题排查为主
*   **自动联调**: 新游上线的第一道安全锁，预设账号白名单，自动联调一次过，实时展示联调日志。
*   **指标诊断**: 拆解复合指标，追溯原始日志和明细。
*   **异常排查**: 定位数据丢失或延迟的具体环节。

### 2.5 分析 Agent -以洞察产出为主
*   **聚合取数**: 快速获取广告标签汇总报表
*   **多维分析**: 实时产出多维度的洞察结论。
*   **回本预测**: 基于历史数据预测未来投放表现与风险，大盘回本模型，首日ROI测算，LTV/ROI倍率预测等。

### 2.6 策略Agent -以智能优化为主
*   **智能回推**: 预测分析下探到广告ROI层级，延迟回推策略。
	**自动暂停**: 根据策略结果，自动暂停低效广告。
	**预算出价**: 根据ROI预测结果，自动调整预算和出价策略。

---

## 3. 交互范式设计 (Interaction Paradigm)
*   **无菜单设计**: 移除侧边栏，采用对话流作为第一优先级。
*   **上下文感知**: AI 能够理解当前用户正在查看的 Agent 视图，并提供相关建议。
*   **状态反馈**: 实时展示各 Agent 的运行负载与告警状态。
*   **风格**: 问答用知识库风格，监控用呼吸灯特效，诊断用侦探风格，分析用数据科学家风格。

---

## 4. 业务流程 (Business Flow)

1.  **问答agent：提交对接需求 -> AI 自动标准化 -> 产研收到通知。
2.  **诊断gent：提审通过后 -> 自动联调 -> 实时展示链路日志。
3.  **监控Agent：实时观测链路 -> 发现异常 -> “诊断 Agent”介入排查。
4.  **分析Agent：产出洞察 -> “预测 Agent”给出预算建议 -> “回传 Agent”调整策略 ->暂停广告。

## 5. 未来规划 (Roadmap)
1.  **媒体mcp集成**: 自动暂停等广告操作。
2.  **移动端驾驶舱**: 开发适配手机端的简易版监控面板和语音交互。
