## 0. 文档目标

这份文档把小乔智投第一阶段核心对象正式落成数据表结构。

职责不是第一次思考数据，而是把前面的对象模型正式定表。

---

## 1. 定表原则

1. 先保证运行时闭环成立
2. 轻重工作流共享主干
3. 结果物独立存储
4. 允许后续扩展更多业务能力

---

## 2. 当前正式表总览

建议第一阶段正式维护以下 `8` 张表：

1. `xiaoqiao_conversations`
2. `xiaoqiao_messages`
3. `xiaoqiao_tasks`
4. `xiaoqiao_task_contexts`
5. `xiaoqiao_workflow_results`
6. `xiaoqiao_evidence_items`
7. `xiaoqiao_case_records`
8. `xiaoqiao_routing_decisions`

---

## 3. 表结构

### 3.1 xiaoqiao_conversations

作用：保存连续对话

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 主键 |
| user_id | string | 用户标识 |
| title | string | 会话标题 |
| status | string | `active / archived / closed` |
| current_mode | string | `natural-chat / light-workflow / heavy-workflow` |
| started_at | datetime | 创建时间 |
| updated_at | datetime | 最近更新时间 |
| last_message_at | datetime | 最后一条消息时间 |

### 3.2 xiaoqiao_messages

作用：保存消息流

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 主键 |
| conversation_id | string | 外键，关联 conversations |
| role | string | `user / assistant / system` |
| message_type | string | `user_input / assistant_reply / clarification / system_notice / workflow_summary` |
| content | text | 消息内容 |
| related_task_id | string nullable | 关联任务 |
| created_at | datetime | 创建时间 |

### 3.3 xiaoqiao_tasks

作用：保存正式任务

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 主键 |
| conversation_id | string | 外键 |
| task_type | string | `help / diagnosis / demand / debugging / monitor / material-analysis / forecast` |
| workflow_level | string | `light / heavy` |
| status | string | `created / clarifying / running / waiting / completed / archived / downgraded` |
| owner_type | string | `xiaoqiao / sub-agent / human-escalation` |
| source_message_id | string nullable | 首条触发消息 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |
| closed_at | datetime nullable | 完结时间 |

### 3.4 xiaoqiao_task_contexts

作用：保存任务上下文

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| task_id | string | 主键兼外键 |
| is_business_related | boolean | 是否业务相关 |
| business_domain | string nullable | 当前阶段主要为 `ad` |
| intent_type | string nullable | `help / diagnosis / demand / debugging / monitor / material-analysis / forecast` |
| media | string nullable | 媒体 |
| app | string nullable | 应用 |
| plan_id | string nullable | 计划标识 |
| device_id | string nullable | 设备标识 |
| time_range | string nullable | 时间范围 |
| attachments_json | json | 附件列表 |
| missing_fields_json | json | 缺失字段对象 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 3.5 xiaoqiao_workflow_results

作用：保存结构化结果

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 主键 |
| task_id | string | 外键 |
| result_type | string | `help_answer / diagnosis_report / demand_form / debugging_report / monitor_snapshot / material_report / forecast_report` |
| summary | text | 结果摘要 |
| structured_payload_json | json | 结构化结果体 |
| confidence | string nullable | `low / medium / high` |
| next_action | text nullable | 下一步动作 |
| created_at | datetime | 创建时间 |

### 3.6 xiaoqiao_evidence_items

作用：保存排查/联调证据

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 主键 |
| task_id | string | 外键 |
| source_type | string | `upload / knowledge / media-data / callback-log / client-log / report` |
| source_name | string | 来源名称 |
| summary | text | 证据摘要 |
| detail | text nullable | 证据详情 |
| happened_at | datetime nullable | 证据发生时间 |
| created_at | datetime | 写入时间 |

### 3.7 xiaoqiao_case_records

作用：保存可复用案例

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 主键 |
| source_task_id | string | 外键 |
| case_type | string | `diagnosis / demand / debugging / monitor / mixed` |
| title | string | case 标题 |
| summary | text | case 摘要 |
| reusable_points_json | json | 可复用要点 |
| status | string | `draft / active / archived` |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 3.8 xiaoqiao_routing_decisions

作用：保存路由判断结果

字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 主键 |
| conversation_id | string | 外键 |
| source_message_id | string | 外键 |
| is_business_related | boolean | 是否业务相关 |
| business_domain | string nullable | `ad / other / null` |
| intent_type | string nullable | 当前意图类型 |
| workflow_level | string nullable | `none / light / heavy` |
| clarification_needed | boolean | 是否需要追问 |
| decision_reason | text | 判定原因 |
| created_at | datetime | 创建时间 |

## 4. 关系说明

### 一对多

1. `xiaoqiao_conversations -> xiaoqiao_messages`
2. `xiaoqiao_conversations -> xiaoqiao_tasks`
3. `xiaoqiao_tasks -> xiaoqiao_workflow_results`
4. `xiaoqiao_tasks -> xiaoqiao_evidence_items`

### 一对一

1. `xiaoqiao_tasks -> xiaoqiao_task_contexts`

### 一对零或一

1. `xiaoqiao_tasks -> xiaoqiao_case_records`
