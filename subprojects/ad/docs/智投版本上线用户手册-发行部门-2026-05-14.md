# 智投系统本次上线用户手册

面向对象：各发行部门、优化师、采购及需要查看投放数据的同事  
上线范围：代理折扣维护、广告账户授权、报表问数与报表筛选能力

## 一、本次上线变化总览

本次上线主要有两类变化：

1. 代理折扣和广告账户授权信息更完整：在维护代理商折扣、绑定广告账户时，新增了付款模式、结算账期等信息，便于按不同结算政策计算折后消耗。
2. 报表问数能力更丰富：更多报表可以在魔方问数中直接提问查询，多个报表页面也增加了优化师筛选，方便按负责人查看投放表现。

## 二、代理折扣维护有什么变化

### 1. 新增需要维护的信息

进入“资产管理 - 代理商管理”后，新增或编辑代理商折扣时，需要关注以下信息：

| 信息 | 说明 |
| --- | --- |
| 媒体名称 | 选择该折扣适用的媒体 |
| 投放模式 | 区分自投、代投等投放方式 |
| 付款模式 | 可选预付、垫付、无 |
| 结算账期 | 可选 1 个月、2 个月、3 个月、无 |
| 日期 | 设置折扣生效的开始日期和结束日期 |
| 折扣 | 填写对应折扣比例 |

### 2. 填写规则

- 付款模式选择“预付”时，页面不需要再填写结算账期。
- 付款模式选择“垫付”时，请按实际政策选择结算账期。
- 同一个代理商在相同媒体、投放模式、付款模式、结算账期和日期范围下，不应重复维护同一条折扣。
- 折扣设置完成后，会影响后续折后消耗计算，请确认日期范围和折扣比例无误后再提交。

### 3. 对业务同事的影响

- 采购同事录入代理折扣时，需要补齐付款模式和结算账期。
- 优化师或发行同事查看投放成本时，系统会按更细的结算政策匹配折扣。
- 对于预付和垫付，折后消耗的计算口径一致：按账面消耗结合返点折算。

## 三、广告账户授权有什么变化

### 1. 绑定代理商时需要补充更多信息

进入“资产管理 - 广告账户授权”，在授权或绑定投放账户时，页面会要求选择：

| 信息 | 说明 |
| --- | --- |
| 项目 | 账户所属项目 |
| 代理商 | 当前广告账户绑定的代理商 |
| 代理模式 | 自投或代投 |
| 付款模式 | 预付、垫付或无 |

部分账户相关信息会继续用于后续折扣匹配和报表计算，请在授权时按真实业务关系选择。

### 2. 操作建议

- 新授权账户请先确认项目、代理商和付款方式，再提交授权。
- 如果账户已授权但代理商或付款方式有变更，请及时更新，避免后续成本口径不一致。
- 暂时无法确认代理商时，不建议随意选择；应先与采购或投放负责人确认。

## 四、魔方问数新增了哪些报表能力

本次上线后，更多报表可以通过魔方问数查询。发行部门同事可以直接用自然语言提问，例如“看昨天某项目的活动表现”“查某优化师最近 7 天的素材消耗和激活成本”。

### 1. 可查询的报表范围

| 分类 | 报表 |
| --- | --- |
| 核心报表 | 概览、实时数据、日周月报 |
| 分析报表 | ROI 分析、LTV 分析、留存分析 |
| 素材与账户 | 广告素材、账户报表 |
| 新增报表 | 活动报表、包体报表、直播报表、关键词报表 |
| 回流报表 | 回流数据、回流 LTV、回流留存 |

说明：“回流 ROI”本次按最新口径调整为“回流留存”。

### 2. 推荐提问方式

可以按下面的方式提问：

- “帮我看昨天 XX 项目的活动报表，按消耗最高排序。”
- “查一下最近 7 天 XX 优化师的广告素材表现。”
- “看 XX 项目本周 ROI 分析，展示 7 日付费次数和 LTV 系数。”
- “查 XX 包体最近 3 天的激活成本和付费情况。”
- “看昨天直播报表里消耗最高的前 5 条。”
- “查某项目最近 7 天回流留存情况。”

### 3. 查询结果展示限制

为了保证查询速度和结果可读性，系统会默认控制展示数量：

- 明细类结果默认优先展示前 10 条。
- 多选筛选项最多选择 10 个。
- 如果结果太多，建议增加项目、日期、媒体、优化师等筛选条件。
- 如果想看整体情况，建议先问“汇总”，再逐步追问明细。

## 五、哪些页面新增了优化师筛选

本次多个报表支持按优化师查看数据，方便发行部门按负责人跟进投放表现。

### 1. 支持优化师筛选的报表

- 日周月报
- 广告素材
- ROI 分析
- LTV 分析
- 留存分析
- 活动报表
- 包体报表
- 直播报表
- 关键词报表

### 2. 暂不支持优化师筛选的报表

- 实时数据
- 回流数据
- 回流 LTV
- 回流留存

如果筛选项里没有看到某个优化师，通常表示当前查询范围内没有匹配到可识别的数据。

## 六、广告素材报表有什么变化

广告素材报表本次同步增强了筛选和展示能力：

- 账户筛选支持多选，最多选择 10 个账户。
- 保留旧版常用筛选习惯，降低切换成本。
- 新增制作方相关筛选，便于按素材责任人或制作来源分析。
- 展示字段补充横竖版、上传人、达人、创意人、制作人等信息。
- 可结合优化师筛选，查看某位优化师负责素材的消耗、激活、付费表现。

## 七、常见问题

### 1. 为什么现在维护折扣时要选付款模式和结算账期？

因为不同媒体和代理商的结算政策会影响折扣匹配。补齐这些信息后，后续计算折后消耗会更准确。

### 2. 预付和垫付的折后消耗口径是否不同？

本次上线后，两种付款模式最终折后消耗口径一致，系统会按账面消耗和返点折算。

### 3. 魔方问数返回的数据太少怎么办？

明细结果默认会控制数量。如果需要更精确结果，可以继续补充筛选条件，例如项目、日期、媒体、账户、优化师、活动或包体。

### 4. 为什么有些报表不能按优化师筛选？

实时数据和回流类报表本次暂不支持优化师筛选。其他已支持的报表可优先使用优化师筛选查看负责人表现。

### 5. 素材页账户为什么最多选 10 个？

为了避免一次查询范围过大影响速度和结果阅读，账户、广告、广告组、活动、包体等多选项最多支持 10 个。

## 八、给发行部门的使用建议

1. 查整体表现时，先问项目或日期汇总，再追问具体账户、素材、活动或包体。
2. 查负责人表现时，优先使用优化师筛选。
3. 查素材问题时，结合制作方、上传人、创意人等字段一起看。
4. 发现折后消耗异常时，先检查广告账户是否绑定了正确代理商、付款模式和折扣日期。
5. 新账户上线前，尽量先完成代理商和付款信息确认，避免后续报表口径偏差。
