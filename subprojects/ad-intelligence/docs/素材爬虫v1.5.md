🧠 四、核心模块2：爆款预测（Virality Model）🔥
🎯 目标：

预测：

这个 Family 会不会“爆”

📊 核心公式（简化版）
Virality Score =
0.3 × growth_rate +
0.25 × seed_activation +
0.2 × embedding_coherence +
0.15 × creative_velocity +
0.1 × early_engagement_proxy
输出：
{
  "family_id": "F001",
  "virality_score": 0.87,
  "confidence": 0.81,
  "prediction": "HIGH POTENTIAL"
}

👉 解释：

0.8 = 极大概率爆

0.5–0.8 = 中等增长
<0.5 = 弱趋势
🧠 五、核心模块3：生命周期预测（Lifecycle Model）🔥
🎯 目标：

预测：

一个 Family 还能活多久

📊 生命周期模型：
Birth → Growth → Peak → Saturation → Decline
特征输入：
growth deceleration
creative exhaustion
saturation level
embedding drift
输出：
{
  "stage": "growth",
  "time_to_peak": "2-4 days",
  "remaining_life": "6-8 days",
  "decay_risk": 0.22
}
🧠 六、核心模块4：峰值预测（Time-to-Peak）🔥
🎯 关键问题：

什么时候最值得投？

方法：
peak = argmax(growth_rate - saturation)
输出：
{
  "time_to_peak": "3 days",
  "peak_confidence": 0.76
}

👉 这是广告投放最核心指标

🧠 七、核心模块5：衰退预测（Risk Model）
🎯 判断：

这个 Family 是否要死了？

信号：

seed 激活下降
embedding 收敛
输出：
{
  "decline_risk": 0.74,
  "reason": "creative exhaustion"
}
