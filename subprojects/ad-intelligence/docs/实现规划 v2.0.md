## Phase 1: 采集基础设施（4-6 周）

### 1.1 多账号采集系统

**目标**：建立分布式广告采集网络，覆盖 70%+ 主流广告池

**核心模块**：

```
src/crawler/
├── account_pool.ts        # 账号池管理
├── persona_engine.ts      # 人格模拟器
├── scheduler.ts           # 调度器
└── collectors/
    ├── feed_collector.ts      # 推荐流采集
    ├── search_collector.ts    # 搜索流采集
    ├── related_collector.ts   # 相关推荐采集
    └── profile_collector.ts   # 账号主页采集
```

**账号人格模型**：
```typescript
interface AccountPersona {
  id: string;
  interestProfile: string[];  // ['SLG', 'RPG', '传奇']
  behavior: {
    scrollSpeed: 'fast' | 'slow' | 'random';
    watchTime: number[];       // [min, max] 秒
    clickRate: number;         // 0-1
    searchBias: string[];
  };
  status: 'active' | 'cooldown' | 'banned';
  lastUsed: Date;
}
```

**调度策略**：
- 人格 × 入口 × 时间 三维调度
- 错峰采集（避免风控）
- 单账号频率限制

**验收标准**：
- [ ] 支持 10+ 账号并行采集
- [ ] 日采集量 500-2000 条
- [ ] 账号风控率 < 5%

### 1.2 Seed 关键词引擎

**目标**：自动发现和维护广告触发词库

**核心模块**：

```
src/seed/
├── seed_manager.ts        # Seed 管理器
├── keyword_extractor.ts   # 关键词提取
├── evolution_engine.ts    # 自进化引擎
└── scoring_model.ts       # Seed 评分模型
```

**Seed 类型**：
```typescript
enum SeedType {
  APP = 'app',           // 应用名（传奇/SLG 等）
  KEYWORD = 'keyword',   // 商业词（爆率/VIP/福利）
  BEHAVIOR = 'behavior', // 行为词（下载/上线/领取）
  AD = 'ad'              // 广告衍生词
}

interface Seed {
  id: string;
  type: SeedType;
  text: string;
  score: number;         // 0-1，发现效率
  category: string[];    // ['SLG', '爆率', '福利']
  createdAt: Date;
  lastUsed: Date;
  discoveryCount: number;
}
```

**自进化逻辑**：
```
采集广告 → OCR/ASR → 提取新词 → 过滤商业词 → 评分 → 入池
                    ↓
                淘汰低分 Seed
```

**验收标准**：
- [ ] 初始 Seed 库 100+
- [ ] 每周自动扩展 20+ 新词
- [ ] 新广告发现率提升 30%

**目标**：视频 → 结构化文本数据

**核心模块**：

```
src/pipeline/
├── downloader.ts        # 视频下载
```

**技术选型**：
- OCR: Tesseract / PaddleOCR
- ASR: Whisper (faster-whisper)
- 帧分析：OpenCV + 场景检测

**输出结构**：
```typescript
interface ParsedVideo {
  videoId: string;
  ocrText: string[];           // 按帧提取
  asrText: string;             // 全文转写

}

interface TimelineSegment {
  start: number;  // 秒
  end: number;
  type: 'hook' | 'gameplay' | 'value' | 'cta';
  ocrText: string[];
  asrText: string;
}
```

**验收标准**：
- [ ] OCR 准确率 > 85%
- [ ] ASR 转写延迟 < 30s/视频
- [ ] Hook 提取准确率 > 80%

### 1.4 Embedding 去重系统

**目标**：广告语义去重 + 相似度计算

**核心模块**：

```
src/embedding/
├── embedder.ts          # 向量生成
├── vector_store.ts      # 向量存储
├── dedup_engine.ts      # 去重引擎
└── similarity.ts        # 相似度计算
```

**技术选型**：
- Embedding 模型：bge-m3 / text-embedding-3-small
- 向量库：Chroma / Qdrant

**去重逻辑**：
```typescript
// 多模态融合 embedding
const adEmbedding = fuse({
  ocr: ocrEmbedding,      // 0.3
  asr: asrEmbedding,      // 0.3
  hook: hookEmbedding,    // 0.2
  structure: structEmbed  // 0.1
});

// 相似度 > 0.9 判定为重复
if (similarity > 0.9) {
  markAsDuplicate();
}
```

**验收标准**：
- [ ] 去重准确率 > 90%
- [ ] 向量检索延迟 < 50ms
- [ ] 重复率从 60% 降至 20%

**核心模块**：

```
src/generation/

├── creative_composer.ts   # 创意组合
```

**生成流程**：
```

       ↓
Template Selection（选择高爆模板）

       ↓
```

**输出结构**：

```typescript
  };
  
  variations: GeneratedAd[];
  recommendedLaunchWindow: number; // 天
}
```

**验收标准**：
- [ ] 生成广告人工评分 > 7/10
- [ ] 变体多样性 > 5 版本
