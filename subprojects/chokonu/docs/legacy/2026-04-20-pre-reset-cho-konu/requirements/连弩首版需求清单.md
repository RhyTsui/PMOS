# 连弩首版需求清单

- 版本：v0.1
- 日期：2026-04-20
- 状态：Draft
- 项目：连弩-AI测试平台 / ChoKoNu

## 1. 核心结论

连弩首版需求只围绕一条主链：

`样本 -> suite -> run -> evaluator -> trace -> report -> version compare`

优先级完全继承 `Must / Should / Could / Not-now`，不重新发明范围。

## 5. Not-now

以下明确不写入首版需求承诺：

1. 通用 Agent Builder
2. 大而全低代码 Workflow 设计器
3. 跨 server 复杂 MCP 组合测试
4. 多 persona 长期会话体系

## 6. 验收规则

首版验收必须满足：

1. P0 主链路全部可运行
2. 四类主战场对象至少各有一套可执行测试能力
3. 单次 run 可生成正式报告
4. 至少支持基础版本对比或已预留标准接口

## 7. 需要主管决策的问题

1. 版本对比是否从 P1 提前到首版强承诺
2. Skill / Workflow 轻接入是否保留在一期
3. 人工复核挂点是否只做结果标记，不做完整工作台
