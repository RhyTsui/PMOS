# ChoKoNu 首版需求清单（人读版）

- 项目：`连弩-AI测试平台 / ChoKoNu`
- 文档层级：`delivery-brief`
- 用途：开发评审、老板沟通、下游交付

## 首版目标

ChoKoNu 首版要先把 AI 应用测试与评测的核心闭环跑通：

`标准 -> 样本 -> 评测集 -> 执行 -> 评估 -> Trace -> 报告 -> 回归`

## 本轮必须做

1. 提供测试标准定义能力。
2. 提供测试样本沉淀能力。
3. 提供评测集组织能力。
4. 提供执行任务配置能力。
5. 提供评估器执行能力。
6. 提供 Trace 归集能力。
7. 提供测试报告生成能力。
8. 提供回归集沉淀能力。
9. 提供版本评审输入能力。

## 本轮建议做

1. 支持版本维度结果对比。
2. 支持对象维度筛选与查看。
3. 支持报告回挂需求与版本。

## 本轮暂不做

1. 通用 AI Agent 开发平台能力。
2. 大而全资源门户。
3. 复杂跨系统实时治理闭环。
