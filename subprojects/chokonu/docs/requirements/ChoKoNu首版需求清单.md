# ChoKoNu 首版需求清单

- 项目：`连弩-AI测试平台 / ChoKoNu`
- 文档层级：`internal-detailed`
- 规则：一条需求一句话，先动作，后对象

## P0

1. 建立测试标准定义入口。
2. 建立测试样本沉淀入口。
3. 建立评测集创建与维护入口。
4. 建立执行任务配置入口。
5. 支持运行评估器并返回结构化结果。
6. 归集失败样本对应的 Trace 信息。
7. 生成可复用的测试报告。
8. 支持把失败样本沉淀为回归集。
9. 输出版本评审所需的核心结论。
10. 区分过程文档与交付文档。

## P1

11. 支持按对象筛选 Case、Suite、Run、Report。
12. 支持按版本对比关键测试结果。
13. 支持查看评估器分类与执行记录。
14. 支持查看异常链路的归因摘要。
15. 支持把报告结论回挂到需求和版本。
16. 支持人读版交付文档导出源稿。

## P2

17. 支持更细粒度的质量门禁规则。
18. 支持更多 AI 应用专项测试方法模板。
19. 支持更完整的跨系统集成治理。
