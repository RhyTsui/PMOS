# ChoKoNu v1 测试规格书

- 项目：`连弩-AI测试平台 / ChoKoNu`

- 日期：`2026-04-28`
- 规格类型：`测试规格书`

## 1. 测试目标

本轮测试目标不是验证真实业务闭环，而是验证：

`前端工作台壳是否可访问、可导航、可查看、可演示、可继续接真数据`

## 2. 测试范围

### 2.1 包含

1. 登录页访问
2. 工作台路由访问
3. 左侧导航切换
4. 顶部项目 / 环境 / 版本切换
5. 各页面列表展示
6. 桌面端详情栏展开 / 收起
7. 移动端详情抽屉
8. 登录页到工作台跳转
9. 构建产物生成

### 2.2 不包含

1. 真实登录认证
2. 真实后端接口联调
3. 真实权限校验
4. 真实任务执行
5. 真实 Trace 回传
6. 真实报告生成
