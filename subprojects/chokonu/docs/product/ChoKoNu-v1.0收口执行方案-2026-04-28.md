`先完成 v1.0，并确保它在 2026-05-10 可上线。`

1. `v1.0` 继续推进，并作为唯一开发主线
2. `v1.1` 暂停实现，只保留规划
3. `v1.2` 暂停实现，只保留规划

`持续可用于 Web UI 自动化和 MCP / Agent 的 AI 应用评测`

这句话不能再继续发散。

1. 平台能登录
2. 平台能按项目组织
3. 能承接 `ui / agent / mcp` 三类核心用例
4. 能看到执行、Trace、报告、复核、门禁
5. 两条主线能讲通：

   - MCP / Agent AI 应用评测
