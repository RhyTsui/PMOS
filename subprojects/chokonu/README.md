# ChoKoNu / 连弩-AI测试平台

`ChoKoNu` 是 PMAIOS 下用于承接 AI 应用测试、评测、自动化回归、结果报告与治理闭环的子项目。

## 当前定位

当前项目主线是：

- AI 应用测试与评测
- 评测集与样本资产沉淀
- 评估器与规则执行
- 自动化测试与回归
- 结果报告、异常 Trace 与治理闭环

当前目录已经从旧 `tracking-acceptance` 语义迁移到：

- `subprojects/chokonu/`
