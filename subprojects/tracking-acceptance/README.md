# AI 埋点验收项目

广告/数据业务的埋点验收自动化系统

## 项目背景

基于 2026-04-10 周报和 04-02/04-03/04-13 内部会议纪要整理。

## 需求背景

### 问题现状

1. **埋点数据上报不规范**
   - 游戏事件字段非标准化，各游戏不一致
   - 数据上报后存在很多问题，需要人工对接和排查
   - 问题根因：底层数据上报不规范、字段缺失、格式错误

2. **测试验证效率低**
   - 当前测试方式：人工录入 → 人工检查 → 人工验证
   - 测试用例需要人工编写，耗时且容易遗漏
   - 问题修复后需要人工回归验证

3. **数据验收流程不透明**
   - 埋点验收状态无法追踪
   - 问题定位困难，需要跨多个系统排查
   - 缺乏统一的验收标准和自动化验收工具

### 会议决策要点

**04-03 会议 - AI 驱动数据上报与测试自动化：**
- 利用 AI 自动生成测试用例和测试数据
- AI 自动识别上报数据与预期格式的差异
- 建立自动化埋点验收流程

**04-03 会议 - AI 驱动数据对接自动化：**
- 游戏运营数据通过 MCP 进行对接
- AI 自动解析数据格式并生成对接文档
- 5 月份计划完成数据对接自动化

**04-13 会议 - 新数仓数据架构与采集优化：**
- 统一采集平台，各业务线数据统一接入
- 解决历史数据缺失和格式不一致问题
- 建立数据质量监控机制
