# Session Start Prompts

这个目录只做一件事：
存放“下次开会话时，第一句话该怎么说”的续跑提示词。

现在按最简单的规则用：

1. 默认只看 `CURRENT.md`
2. 这份文档是滚动文档，用过就清理或直接改写
3. 不做复杂归档，不要求保留多版状态流
4. 只有在确实需要保留另一条独立续跑线时，才在 `CURRENT.md` 里分段写开

推荐写法尽量固定成四块：

1. `触发短语`
2. `当前有效判断`
3. `下次回来先做什么`
4. `可直接复制的启动提示词`

如果某一段已经失效：

1. 直接删掉
2. 或在该段标题下写明 `已作废` 和替代内容

重点不是留历史，而是让下一次打开会话时能立刻续跑。
