# AI埋点平台概念图评选 v1

日期：2026-04-30

## 1. 评选目标

为本轮设计图与前端交互提供统一概念方向，不再开放多风格并行发散。

## 2. 候选方向

### A. 企业级数据运营后台

关键词：

- 浅底深字
- 表格优先
- 高信息承载
- 强筛选
- 强状态
- 弱装饰

判断：

- 最适合当前平台

### B. 品牌展示型控制台

关键词：

- 大卡片
- 大插图
- 强品牌色
- 叙事型概览

判断：

- 不适合当前平台

### C. 科技炫光大屏

关键词：

- 深色底
- 视觉冲击强
- 图表优先
- 装饰性强

判断：

- 不适合当前平台

## 3. 评选结论

本轮正式选定：

- A. 企业级数据运营后台

## 4. 设计约束

- 概览页中密
- 列表页中高密
- 日志 / 规则 / 报告表默认紧凑优先
- 强调角色作用域，不做全员同屏信息平铺
- 图表只服务判断，不喧宾夺主
- 表格体验优于 Excel 阅读体验
- AI 助手不是泛聊天入口，需明确包含 `AI Coding` 子能力
- `AI Coding` 的定义固定为：`知识库 + MCP + 代码仓库`
