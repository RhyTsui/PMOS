类型与格式规则

  • 时间字段统一 13 位毫秒时间戳。
  • 步骤 ID / 事件 ID 统一使用数字字符串或整型，不混用。
  • map/json 字段必须同时说明 key 和 value 语义。
  • 枚举字段必须能回到字典或内容 ID。
