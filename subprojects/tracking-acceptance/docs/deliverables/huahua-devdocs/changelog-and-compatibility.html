变更与兼容

花花世界当前最突出的版本问题不是 SDK 版本，而是步骤版本、事件迁移和字段来源变化。

步骤版本

step_version说明
20250122旧版前置 / 新手链路版本
20250724新版前置链路版本，新增设备信息、服务器列表、网关等步骤

事件迁移

RM 中已明确多组自定义事件是从旧编号迁移而来，例如：

  • 20001 -> 原 10001
  • 20002 -> 原 10002
  • 20004 -> 原 10004
  • 20012 -> 原 10014
  • 20018 -> 原 10031

兼容策略

  • 步骤版本变化必须增量更新 step_version，不能静默替换。
  • 自定义事件迁移必须保留新旧映射。
  • 字段命名变更必须同步更新字段参考和验收规则。
  • 当前未补完字段的事件必须显式标记“待补完”，不能伪装成稳定规范。
