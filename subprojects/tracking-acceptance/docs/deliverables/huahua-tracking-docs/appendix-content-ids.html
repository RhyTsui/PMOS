为什么不把 4396 行 ID 全塞进正文

`内容ID字典0.8.4` 是项目级对象字典，不是接入主册的正文部分。它的作用是：

  • 给 `tutorial_id`、`grow_target`、`unlock_target`、`currency_id`、`plant_id`、`goods_id` 等字段提供对象 ID 真相源。
  • 为玩法对象、商店对象、植物对象、配方对象、活动对象、扭蛋对象等提供统一编号。
  • 作为后续规则生成和验收解释时的补充字典。

文档策略：主文档只说明“哪些字段依赖内容 ID 字典”，不在主册正文中内嵌 4396 行字典明细，以免破坏主文档可读性。
