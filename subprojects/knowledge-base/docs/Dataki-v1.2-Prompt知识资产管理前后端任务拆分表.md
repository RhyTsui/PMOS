# Dataki v1.2 Prompt 知识资产管理前后端任务拆分表
- 版本：v1.2
- 日期：2026-04-28
- 状态：Draft
- 适用范围：Dataki v1.2 Prompt 开发拆分
- 读者对象：产品、前端、后端、测试、AI

## 1. 拆分原则

1. 先完成 Prompt 一级菜单下的最小闭环，再考虑增强能力。
2. 先完成列表、详情、编辑、共享、绑定五类 P0 页面，再扩展调用记录等 P1 页面。
3. Prompt 在导航上独立，在权限上复用知识库共享机制。
4. 所有任务都必须能回挂到 v1.2 的 Prompt 补充需求文档。

## 2. 前端任务

| 模块 | 任务 | 优先级 | 说明 |
| --- | --- | --- | --- |
| 导航入口 | 新增一级菜单 `Prompt` | P0 | 与知识库、智能体并列 |
| Prompt 列表 | 新增 Prompt 一级菜单列表页 | P0 | 沿用原系统后台列表布局 |
| Prompt 列表 | 支持关键词、共享范围、状态筛选 | P0 | 不增加复杂高级筛选 |
| Prompt 详情 | 新增 Prompt 资产详情页 | P0 | 展示正文、变量说明、已共享空间、已挂载对象 |
| Prompt 编辑 | 新增 Prompt 新建 / 编辑页 | P0 | 单页表单 |
| Prompt 共享 | 新增共享弹层 | P0 | 与知识库共享到空间机制一致 |
| Prompt 绑定 | 新增绑定弹层或右侧抽屉 | P0 | 用于主对话框、业务 Chat、外部 Agent |
| 状态反馈 | 新增空态、无权限态、保存成功、共享成功、绑定成功反馈 | P0 | 轻量反馈即可 |

## 3. 后端任务

| 模块 | 任务 | 优先级 | 说明 |
| --- | --- | --- | --- |
| 资产模型 | 增加 Prompt 资产类型 | P0 | 与统一资产模型对齐 |
| Prompt 模型 | 新增 Prompt 专属字段定义 | P0 | 名称、用途说明、正文、变量说明、状态等 |
| Prompt 列表 | 提供 Prompt 列表查询接口 | P0 | 支持搜索、共享范围、状态 |
| Prompt 详情 | 提供 Prompt 详情查询接口 | P0 | 返回已共享空间与挂载对象 |
| Prompt 创建 | 提供 Prompt 新建接口 | P0 | 支持草稿保存 |
| Prompt 更新 | 提供 Prompt 更新接口 | P0 | 区分编辑与发布动作 |
| Prompt 共享 | 提供共享 / 取消共享 / 更新共享权限接口 | P0 | 复用知识库共享到空间口径 |
| Prompt 绑定 | 提供绑定 / 解绑接口 | P0 | 面向主对话框、业务 Chat、外部 Agent |
| 调用校验 | 增加共享、绑定、权限三级校验 | P0 | 未共享、未绑定或无权限不得调用 |
| 调用记录 | 记录调用主体与调用时间 | P0 | 至少支持 user/chat/agent/app |

## 4. 测试任务

| 模块 | 任务 | 优先级 | 说明 |
| --- | --- | --- | --- |
| 菜单结构 | 验证 Prompt 是否为一级菜单 | P0 | 与知识库、智能体并列 |
| 列表页 | 验证搜索、共享范围筛选、状态筛选 | P0 | 核心入口 |
| 详情页 | 验证详情字段、共享空间、挂载对象展示 | P0 | 核心详情 |
| 编辑页 | 验证新建、保存草稿、编辑更新 | P0 | 必填校验生效 |
| 共享流 | 验证共享、取消共享、权限更新 | P0 | 必须与知识库共享行为一致 |
| 绑定流 | 验证绑定、解绑、状态回写 | P0 | 绑定对象范围正确 |
| 调用链路 | 验证未共享、未绑定、无权限时拦截 | P0 | 核心风险点 |

## 5. 当前结论

Prompt 在 Dataki v1.2 中的开发拆分口径已经明确：
1. 导航层面独立成一级菜单。
2. 协作层面复用空间共享机制。
3. 研发层面先完成列表、详情、编辑、共享、绑定五类 P0 闭环。
