# Dataki v1.2 Prompt 知识资产管理设计执行简版
- 版本：v1.2
- 日期：2026-04-28
- 状态：Current
- 适用对象：设计师、产品、AI 出图执行者

## 1. 本次要出什么

本次只做 `Prompt 知识资产管理` 设计出图，不改前端，不扩产品框架。

Prompt 在 Dataki v1.2 中定位为：
- 一级菜单
- 与知识库、智能体并列
- 一种正式资产
- 使用和知识库一致的空间共享机制

不是：
- Prompt 中台
- Prompt 工程平台
- Prompt 评测平台
- Prompt 工作流平台

## 2. 视觉基准

统一参考这张图：
- [d67cfc1b-35a4-4d0e-a10f-b8e3dce0f8be.png](/E:/AI/ai-os/docs/sources/inbox/d67cfc1b-35a4-4d0e-a10f-b8e3dce0f8be.png)

必须沿用的语义：
- Dataki 品牌标识
- `链接知识与数据`
- 蓝紫渐变品牌语言
- 干净、克制、留白充足的企业后台风格

## 3. 结构红线

1. Prompt 是一级菜单。
2. Prompt 与知识库、智能体并列。
3. 不新增新的系统框架。
4. 不新增独立 Prompt 首页。
5. 不新增实验、评测、编排、运营大盘。
6. 不再使用部门权限口径，统一改为空间共享口径。

## 4. 必出页面

1. Prompt 一级菜单列表页
2. Prompt 资产详情页
3. Prompt 新增 / 编辑页
4. Prompt 共享弹层
5. Prompt 绑定弹层或右侧抽屉

## 5. 每页必须有什么

### 5.1 Prompt 一级菜单列表页

必须有：
- 左侧并列菜单：知识库、Prompt、智能体
- 页面标题 `Prompt 资产管理`
- 搜索框
- 空间或共享范围筛选
- 状态筛选
- 新增按钮
- 列表列：名称、用途说明、归属、负责人、状态、最近更新时间

### 5.2 Prompt 资产详情页

必须有：
- 名称
- 用途说明
- 正文内容
- 变量说明
- 适用模型或场景
- 归属
- 负责人
- 状态
- 版本说明
- 最近更新时间
- 已共享空间
- 已挂载对象

### 5.3 Prompt 新增 / 编辑页

必须有：
- 名称输入
- 用途说明输入
- 正文内容编辑区
- 变量说明区
- 适用模型或场景输入
- 负责人
- 状态切换
- 保存按钮
- 发布按钮

### 5.4 Prompt 共享弹层

必须有：
- 空间选择
- 已共享空间列表
- 共享权限选择
- 确认共享按钮
- 取消共享动作

### 5.5 Prompt 绑定弹层 / 抽屉

必须有：
- 绑定对象类型
- 对象列表
- 当前绑定状态
- 权限说明
- 确认绑定按钮

## 6. 文案口径

统一用：
- `Prompt 资产管理`
- `Prompt 详情`
- `新建 Prompt`
- `编辑 Prompt`
- `共享到空间`
- `绑定到主对话框`
- `绑定到业务 Chat`
- `绑定到外部 Agent`

不要写成：
- Prompt 中台
- Prompt 平台
- Prompt 工程中心
- Prompt 运营中心

## 7. 设计师一句话任务

请在 Dataki 原系统后台布局基础上，新增一个与知识库、智能体并列的 `Prompt` 一级菜单，并出对应的 Prompt 资产列表、详情、编辑、共享和绑定页面设计图，权限口径统一使用空间共享机制。

要求：

## 9. 验收标准

1. 一眼看上去还是 Dataki，不是新系统。
2. 一眼看上去 Prompt 与知识库、智能体并列。
3. 页面数量克制，没有额外膨胀。
4. 页面结构清楚，可以直接进入前端切图和实现。
