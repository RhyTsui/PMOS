# 知识资产平台 v1.1 页面清单

- 版本：v1.1
- 日期：2026-04-20
- 状态：规划中

## 1. 页面结构说明

v1.1 页面设计服务于“框架搭建”目标，重点覆盖：

1. 接入
2. 资产
3. 消费
4. 广场
5. 治理

不追求页面数量完整，而追求平台骨架完整。

## 2. 一级导航

建议一级导航：

1. 接入中心
2. 知识资产
3. 知识消费
4. 知识广场
5. 资产治理

## 3. 页面清单

### 3.1 接入中心

#### 页面 1：接入列表页

用途：

- 查看所有接入任务
- 查看接入来源、状态、更新时间

核心模块：

- 来源类型筛选
- 状态筛选
- 最近更新时间
- 新建接入按钮

#### 页面 2：新建接入页

用途：

- 创建新的知识接入任务

核心模块：

- 来源类型选择
- 导入方式选择
- 资产归属设置
- 权限设置
- 标签与业务域设置

#### 页面 3：接入详情页

用途：

- 查看某次接入任务的处理结果

核心模块：

- 原始输入预览
- 解析状态
- 结构化抽取结果
- 入库结果
- 错误记录

### 3.2 知识资产

#### 页面 4：资产列表页

用途：

- 浏览平台内所有知识资产

核心模块：

- 搜索框
- 类型筛选
- 来源筛选
- 标签筛选
- 业务域筛选
- 权限范围筛选

#### 页面 5：资产详情页

用途：

- 查看单个资产的完整信息

核心模块：

- 原文内容
- 结构化摘要
- 标签与来源
- 权限与责任人
- 版本与更新时间
- 引用关系
- 修正记录

#### 页面 6：会议纪要详情页

用途：

- 展示会议纪要资产的结构化信息

核心模块：

- 原始纪要
- 摘要
- 参与方
- 决策项
- 待办项
- 待确认问题
- 权限设置

### 3.3 知识消费

#### 页面 7：消费配置页

用途：

- 配置知识资产如何被不同消费方使用

核心模块：

- 消费方类型选择
- 知识范围绑定
- 资产类型过滤
- 权限校验说明

#### 页面 8：平台内 Chat 演示页

用途：

- 演示平台内 Chat 调用知识资产

核心模块：

- 会话输入区
- 引用片段展示
- 来源展示
- 反馈入口

#### 页面 9：业务 Chat 绑定页

用途：

- 配置数据分析 Chat 与广告业务 Chat 的知识绑定

核心模块：

- Chat 类型切换
- 知识域绑定
- 类型与标签过滤
- 测试调用入口

### 3.4 知识广场

#### 页面 10：知识广场首页

用途：

- 提供知识发现与订阅入口

核心模块：

- 推荐资产
- 热门资产
- 专题集合
- 按类型浏览

#### 页面 11：广场资产详情页

用途：

- 查看可订阅资产详情

核心模块：

- 资产摘要
- 来源说明
- 适用范围
- 权限说明
- 一键订阅/加入个人范围

### 3.5 资产治理

#### 页面 12：权限与引用页

用途：

- 查看和配置资产治理信息

核心模块：

- 权限模板
- 可见/可引用/可编辑设置
- 引用记录
- 消费方绑定记录

#### 页面 13：修正与确认页

用途：

- 管理资产修正与确认状态

核心模块：

- 摘要修正
- 标签修正
- 结构化字段修正
- 错误标记
- 过时标记

### P1 页面

1. 业务 Chat 绑定页
2. 知识广场首页
3. 广场资产详情页
4. 权限与引用页

### P2 页面

1. 修正与确认页
2. 接入详情页增强

## 5. 页面结论

v1.1 页面设计要先把平台骨架立起来：

- 接入有入口
- 资产有目录
- 消费有挂载
- 广场有发现
- 治理有控制

页面数量可以少，但结构不能缺。
