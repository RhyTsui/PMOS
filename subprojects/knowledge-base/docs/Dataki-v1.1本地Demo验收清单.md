# Dataki v1.1 本地 Demo 验收清单
- 版本：v1.1
- 日期：2026-04-23
- 状态：Draft
- 适用范围：Dataki v1.1 本地 Demo 验收
- 读者对象：产品、前端、后端、测试、AI

## 1. 环境验收

1. WeKnora 容器栈可启动
2. 前台、后端、数据库、检索依赖可访问
3. Dataki 品牌资源已生效

## 2. 账号与组织验收

1. 无超管时可完成首次注册
2. 首个用户注册后自动成为超管
3. 普通用户只可通过小闪扫码登录或小闪手机号登录
4. 小闪用户字段可正确同步
5. 小组 / 部门 / 公司 / 集团信息展示正确

## 3. 品牌验收

1. 产品名已统一为 Dataki
2. Slogan 已统一为 让知识与数据相遇
3. LOGO 与必要品牌资源已替换
4. 页面视觉符合 Dataki 品牌包要求

## 4. 统一配置验收

1. 超管可进入高级设置页
2. 超管可配置统一大模型服务
3. 超管可配置统一 MCP 服务
4. 普通用户不可修改上述配置
5. 普通用户只看到统一下发结果

## 5. MCP 验收

1. 默认内置数据部 MCP 可正常展示
2. 超管可管理 MCP 外挂
3. 普通用户不会看到不该暴露的复杂配置

## 6. 权限验收

1. 超管是唯一系统级配置入口
2. 普通用户默认采用统一配置
3. 用户管理能力可用
4. 用户组织信息按小闪结构正确映射

## 7. 回归验收

1. 导入、解析、检索链路未被 Dataki 改造破坏
2. 原有聊天基础链路仍可工作
3. 容器环境重启后状态稳定

## 8. 演示通过标准

Demo 最低通过线：

1. 能登录
2. 能看到 Dataki 前台
3. 超管能完成统一配置
4. 普通用户只能通过小闪扫码 / 手机号登录
5. 能展示统一模型名称与默认 MCP 状态
