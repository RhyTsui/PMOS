# Dataki v1.2 Prompt 知识资产管理页面清单与页面流
- 版本：v1.2
- 日期：2026-04-28
- 状态：Draft
- 适用范围：Dataki v1.2 Prompt 页面 IA 与页面流
- 读者对象：产品、设计、前端、后端、测试、AI

## 1. 页面结构原则

Dataki v1.2 的 Prompt 页面只解决一件事：
- 让 Prompt 作为一级菜单模块，进入原系统后台结构，并完成管理、共享、绑定、调用闭环

因此本次页面结构原则是：
1. Prompt 是一级菜单。
2. Prompt 与知识库、智能体并列。
3. 页面沿用原系统后台布局，不额外新起平台框架。
4. 页面只覆盖列表、详情、编辑、共享、绑定五类最小闭环页面。

## 2. 页面入口建议

登录后一级导航建议至少包含：
1. 知识库
2. Prompt
3. 智能体

说明：
1. Prompt 在导航层级上独立出现。
2. Prompt 在权限与共享机制上继续复用知识库逻辑。
3. 不新增新的系统级框架。

## 3. 页面清单

### 3.1 Prompt 模块

#### 页面 1：Prompt 一级菜单列表页

用途：
- 浏览平台内所有 Prompt 资产
- 按共享范围、状态、关键词筛选 Prompt
- 进入详情或编辑

核心模块：
- 页面标题
- 搜索框
- 共享范围筛选
- 状态筛选
- 新增 Prompt 按钮
- 列表区

#### 页面 2：Prompt 资产详情页

用途：
- 查看单个 Prompt 资产完整信息
- 查看已共享空间
- 查看已挂载到哪些消费端
- 从详情发起编辑、共享或绑定

#### 页面 3：Prompt 新增 / 编辑页

用途：
- 新建 Prompt 资产
- 修改 Prompt 资产内容与配置

#### 页面 4：Prompt 共享弹层

用途：
- 把 Prompt 共享到指定空间
- 查看当前已共享空间
- 调整共享权限或取消共享

#### 页面 5：Prompt 绑定弹层 / 右侧抽屉

用途：
- 绑定 Prompt 到主对话框、业务 Chat 或外部 Agent
- 查看当前绑定状态
- 校验绑定权限与空间可用性

## 4. 核心页面流

### 4.1 Prompt 新增流

1. 用户进入 `Prompt 一级菜单列表页`
2. 点击 `新增 Prompt`
3. 进入 `Prompt 新增 / 编辑页`
4. 填写名称、用途说明、正文内容、变量说明、适用模型或场景
5. 保存为草稿或直接发布
6. 保存成功后回到 `Prompt 资产详情页` 或 `Prompt 一级菜单列表页`

### 4.2 Prompt 查看流

1. 用户进入 `Prompt 一级菜单列表页`
2. 搜索或筛选目标 Prompt
3. 点击某条 Prompt
4. 进入 `Prompt 资产详情页`
5. 查看正文、变量说明、状态、版本说明、已共享空间、已挂载对象

### 4.3 Prompt 编辑流

1. 用户进入 `Prompt 资产详情页`
2. 点击 `编辑`
3. 进入 `Prompt 新增 / 编辑页`
4. 修改字段内容
5. 点击保存
6. 返回 `Prompt 资产详情页`

### 4.4 Prompt 共享流

1. 用户进入 `Prompt 资产详情页`
2. 点击 `共享`
3. 打开 `Prompt 共享弹层`
4. 选择目标空间
5. 选择共享权限
6. 点击确认共享
7. 共享成功后回写到详情页的共享空间区

补充约束：
- 共享机制与知识库保持一致
- 同一 Prompt 可共享到多个空间
- 成员最终权限仍受空间角色限制

### 4.5 Prompt 绑定流

1. 用户进入 `Prompt 资产详情页`
2. 点击 `绑定`
3. 打开 `Prompt 绑定弹层 / 右侧抽屉`
4. 选择绑定对象类型
5. 选择具体对象
6. 查看权限说明
7. 点击确认绑定
8. 绑定成功后回写到详情页的挂载对象区

补充约束：
- 未共享到目标空间的 Prompt 不得被该空间内业务对象绑定使用
- 未绑定的消费端不得调用 Prompt

### 4.6 Prompt 调用流

1. 主对话框、业务 Chat 或外部 Agent 发起 Prompt 调用
2. 系统先检查该 Prompt 是否已共享到当前可访问空间
3. 系统再检查该消费端是否已绑定此 Prompt
4. 系统最后检查当前主体是否具备绑定 / 调用权限
5. 校验通过后允许调用
6. 记录调用主体与调用时间

## 5. 页面优先级

### P0

1. Prompt 一级菜单列表页
2. Prompt 资产详情页
3. Prompt 新增 / 编辑页
4. Prompt 共享弹层
5. Prompt 绑定弹层 / 右侧抽屉

### P1

1. Prompt 版本详情增强
2. Prompt 调用记录页

## 6. 测试关注点

1. Prompt 是否以一级菜单出现
2. Prompt 是否与知识库、智能体并列
3. 列表、详情、编辑、共享、绑定五个页面是否闭环
4. 共享与绑定是否按空间口径生效
5. 未共享、未绑定或无权限时是否被正确拦截

## 7. 页面结论

Prompt 知识资产管理的页面量不需要多，但结构必须完整：
- 有一级菜单入口
- 有列表
- 有详情
- 有编辑
- 有共享
- 有绑定
- 有调用闭环
