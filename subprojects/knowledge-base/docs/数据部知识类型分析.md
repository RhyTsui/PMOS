## 一、知识类型总览

数据部业务知识可分为 **6 大类**：

```
┌─────────────────────────────────────────────────────────┐
│                    数据部知识体系                        │
├─────────────────────────────────────────────────────────┤
│  1. 业务域知识      │  4. 技术域知识                      │
│     - 数据服务业务   │     - API 文档                       │
│     - 广告服务业务   │     - 技术规范                       │
├─────────────────────────────────────────────────────────┤
│  2. 数据域知识      │  5. 流程域知识                      │
│     - 指标定义      │     - 需求流程                       │
│     - 报表口径      │     - 交付流程                       │
├─────────────────────────────────────────────────────────┤
│  3. 系统域知识      │  6. 案例域知识                      │
│     - 数仓架构      │     - 客户案例                       │
│     - 数据血缘      │     - 最佳实践                       │
└─────────────────────────────────────────────────────────┘
```

## 二、详细知识类型

### 1. 业务域知识

#### 1.1 数据服务业务
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 产品说明 | 数据产品的功能和定位 | 报表平台、数据工具、BI 系统 |
| 服务模式 | 如何提供服务 | 需求接入方式、交付周期、支持渠道 |
| 客户信息 | 服务对象和需求 | 各业务部门、外部客户 |
| 收费标准 | 服务定价和资源消耗 | 数据调用量、计算资源 |

#### 1.2 广告服务业务
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 媒体平台 | 各媒体广告平台介绍 | 巨量引擎、腾讯广告、快手 |
| 投放流程 | 广告投放的操作流程 | 开户→充值→创建→优化 |
| 优化策略 | 投放优化方法 | 出价策略、定向优化、创意优化 |
| 账户管理 | 账户结构和权限 | 计划/单元/创意层级 |

---

### 2. 数据域知识（核心）

#### 2.1 指标定义
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 基础指标 | 原始数据指标 | 展示数、点击数、消耗、激活数 |
| 衍生指标 | 计算得出的指标 | CTR、CVR、ROI、ARPU |
| 复合指标 | 多指标组合 | 留存 ROI、LTV、付费率 |
| 指标口径 | 计算逻辑和边界 | 是否去重、时间范围、归因规则 |

#### 2.2 报表口径
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 报表类型 | 各类报表的用途 | 日报、周报、实时报表、专题报表 |
| 数据来源 | 报表数据的来源表 | 广告消耗表、激活事件表 |
| 更新频率 | 数据更新周期 | 小时级、天级、实时 |
| 归因逻辑 | 转化归因规则 | 末次点击、首次点击、多触点 |
| 回推规则 | 历史数据更新规则 | 延迟转化回推、数据修正 |

---

### 3. 系统域知识（核心）

#### 3.1 数仓架构
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 分层结构 | 数仓分层设计 | ODS→DWD→DWS→ADS |
| 主题域 | 数据主题划分 | 广告域、用户域、事件域 |
| 数据流向 | 数据流转路径 | 采集→清洗→聚合→应用 |
| 技术栈 | 使用的技术组件 | Spark、Flink、ClickHouse |

#### 3.2 数据血缘（可解释性核心）
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 表血缘 | 表的上下游依赖 | 表 A 来自表 B 和表 C 的聚合 |
| 字段血缘 | 字段的来源和转换 | 字段 X = 字段 Y / 字段 Z |
| 任务血缘 | ETL 任务的依赖关系 | 任务 A 完成后才能执行任务 B |
| 指标血缘 | 指标的计算链路 | ROI → 消耗表 + 付费表 |

#### 3.3 表结构与字段
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 表定义 | 表的用途和结构 | ad_consumption_day (广告消耗日表) |
| 字段定义 | 字段的含义和类型 | campaign_id (计划 ID, string) |
| 枚举值 | 字段的取值范围 | status: 0-暂停，1-运行，2-删除 |
| 关联关系 | 表之间的关联键 | ad_id JOIN creative.id |

#### 3.4 ETL 流程
| 知识项 | 内容 | 示例 |
|--------|------|------|
| 调度规则 | 任务调度时间和依赖 | 每日 2:00 执行，依赖上游完成 |
| 数据质量 | 校验规则和监控 | 空值检测、波动告警 |
| 异常处理 | 失败处理和恢复 | 重试策略、告警通知 |

## 三、知识类型与检索方式映射

| 知识类型 | 适合检索方式 | 存储建议 |
|----------|-------------|----------|
| 指标定义 | 精确查询 + 语义搜索 | 术语表 + 向量库 |
| 报表口径 | 精确查询 + 图谱溯源 | 图谱 + 文档 |
| 数据血缘 | 图谱查询 | 知识图谱 |
| 表结构 | 精确查询 | 数据库/文档 |
| API 文档 | 语义搜索 + 代码检索 | 向量库 + 代码索引 |
| 流程规范 | 语义搜索 | 向量库 |
| 客户案例 | 语义搜索 + 分类浏览 | 向量库 + 标签 |

## 四、知识结构化程度分析

| 知识类型 | 结构化程度 | 处理方式 |
|----------|-----------|----------|
| 指标定义 | ⭐⭐⭐⭐⭐ 高 | 结构化录入（数据库） |
| 表结构 | ⭐⭐⭐⭐⭐ 高 | 自动同步（元数据管理） |
| 数据血缘 | ⭐⭐⭐⭐ 高 | 图谱建模 |
| API 文档 | ⭐⭐⭐⭐ 高 | 结构化 + 文档 |
| 报表口径 | ⭐⭐⭐ 中 | 文档 + 结构化 |
| 流程规范 | ⭐⭐ 中低 | 文档为主 |
| 客户案例 | ⭐⭐ 中低 | 文档 + 标签 |
| 最佳实践 | ⭐ 低 | 文档 + 向量检索 |

## 五、对知识库设计的启示

### 5.1 存储策略

```
高结构化知识（指标/表结构/血缘）
    → 结构化存储（数据库/图谱）+ 精确查询

中结构化知识（API 文档/报表口径）
    → 混合存储（文档 + 结构化字段）+ 混合检索

低结构化知识（案例/最佳实践）
    → 向量存储 + 语义搜索
```

### 5.2 检索策略

| 查询类型 | 路由策略 |
|----------|----------|
| "XX 指标是什么" | 术语表精确查询 |
| "XX 表有哪些字段" | 元数据查询 |
| "XX 指标怎么算的" | 血缘图谱查询 |
| "XX API 怎么用" | 文档检索 + 代码示例 |
| "类似案例有吗" | 向量语义搜索 |

### 5.3 应用框架

```
┌─────────────────────────────────────────────────────────┐
│                    应用层                                │
│   Chat 问答  /  指标查询  /  血缘追溯  /  API 检索         │
├─────────────────────────────────────────────────────────┤
│                    路由层                                │
│   意图识别 → 查询分类 → 路由到对应检索引擎               │
├─────────────────────────────────────────────────────────┤
│                    检索层                                │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐     │
│  │  术语查询   │  │  图谱查询    │  │  向量检索   │     │
│  │  (指标)     │  │  (血缘)      │  │  (文档)     │     │
│  └─────────────┘  └─────────────┘  └─────────────┘     │
├─────────────────────────────────────────────────────────┤
│                    存储层                                │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐     │
│  │  术语表     │  │  知识图谱    │  │  向量库     │     │
│  │  (MySQL)    │  │  (Neo4j)     │  │  (ChromaDB) │     │
│  └─────────────┘  └─────────────┘  └─────────────┘     │
└─────────────────────────────────────────────────────────┘
```
