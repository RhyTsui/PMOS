# 知识资产平台 v1.1 研发任务单

- 版本：v1.1
- 日期：2026-04-20
- 状态：执行拆解

## 1. 任务目标

本任务单用于把 `v1.1` 规划文档转成研发执行单。

拆解原则：

- 围绕框架搭建
- 围绕三大主场景
- 围绕 Demo 可演示
- 不扩张到非首版深能力

## 2. 任务分组

### A. 底座集成

#### A1. WeKnora 部署验证

目标：

- 跑通最小可用环境

交付：

- 部署说明
- 本地可用环境
- 基础检索验证结果

#### A2. WeKnora 能力映射

目标：

- 明确底座能力与平台对象的映射关系

交付：

- 检索能力映射
- 引用能力映射
- 导入能力映射

### B. 平台模型

#### B1. 资产对象定义

目标：

- 定义统一资产对象

交付：

- `asset`
- `asset_type`
- `source`

#### B2. 权限对象定义

目标：

- 定义统一权限对象

交付：

- `permission_policy`
- 模板权限结构

#### B3. 消费绑定对象定义

目标：

- 定义消费方与资产绑定关系

交付：

- `consumer_binding`
- 绑定规则

### C. 接入链路

#### C1. 文本接入

目标：

- 支持手动文本入库

#### C2. 文档接入

目标：

- 支持文档上传后解析入库

#### C3. 会议纪要文本接入

目标：

- 支持会议纪要文本结构化入库

#### C4. 企业样例来源接入

目标：

- 支持内部文档样例与项目系统样例入库

### D. 资产展示链路

#### D1. 资产列表页

目标：

- 展示和筛选资产

#### D2. 资产详情页

目标：

- 展示原文、摘要、来源、权限、引用

#### D3. 会议纪要详情页

目标：

- 展示纪要结构化内容

### E. 消费链路

#### E1. 平台内 Chat 接入

目标：

- 平台内 Chat 能按权限消费知识

#### E2. 数据分析 Chat 接入

目标：

- 支持数据分析 Chat 绑定知识资产

#### E3. 广告业务 Chat 接入

目标：

- 支持广告业务 Chat 绑定知识资产

### F. 治理链路

#### F1. 权限配置

目标：

- 支持可见/可引用/可编辑

#### F2. 修正能力

目标：

- 支持摘要/标签/纪要字段修正

#### F3. 引用记录

目标：

- 查看资产被哪些消费方使用

### G. 广场链路

#### G1. 广场首页

目标：

- 展示推荐、热门、专题

#### G2. 订阅动作

目标：

- 支持加入个人知识范围

## 5. 验收口径

研发任务验收不按“页面做完”判断，而按链路判断：

1. 接入链路是否成立
2. 资产治理是否成立
3. 消费链路是否成立
4. 三大主场景是否可演示

## 6. 结论

这份任务单的作用是把 `v1.1` 从规划文档推进到研发执行层。

优先保证：

- 底座接通
- 资产模型成立
- 权限链路成立
- Chat 消费成立

而不是先追求复杂连接器或深度运营能力。
