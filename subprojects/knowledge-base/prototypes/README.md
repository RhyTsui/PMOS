## Knowledge Asset Platform UI Prototypes

Three tone-setting prototypes for the `knowledge asset list` page:

1. `asset-list-tone-a.html`
   Editorial workspace. Strong hierarchy, lighter dashboard feeling.
2. `asset-list-tone-b.html`
   Operational console. Dense controls, stronger platform/admin feel.
3. `asset-list-tone-c.html`
   Knowledge atlas. More thematic and exploratory, biased toward discovery.

Open any file directly in a browser to review the visual direction.
