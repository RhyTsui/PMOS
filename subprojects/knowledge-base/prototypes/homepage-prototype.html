平台能做什么

首页只讲清楚，不堆宣传

统一管理知识库

支持企业、部门和项目知识库，也支持个人知识作为补充层，并支持提示词分类管理。

  • 企业、部门、项目是主知识范围
  • 个人知识作为补充和沉淀入口

持续接入内容

会议纪要、小闪文档、小闪聊天记录、外部知识库和订阅内容都可以持续进入平台。

  • 支持文档、图片、视频、链接、脚本包
  • 支持会议 URL 接入

让 AI 直接使用知识

知识不只是存着看，还可以直接接给 Chat、Agent、工作流和 MCP 使用。

  • 支持业务补强和问答
  • 支持知识洞察和分析

按账号控制权限

以小闪账号为主做登录和权限控制，每个知识库可单独设置查看和使用范围。

  • 支持小闪扫码和手机号登录
  • 支持按库配置权限

主对话框预览

首页给轻量预览，不喧宾夺主
统一 AI 工作入口 支持文字 / 语音 / 图片 / 文件 / 视频 / 会议 URL
你可以直接问团队知识、项目知识、公司知识，也可以同时引用多个知识库做分析。 个人知识支持按需挂载，不作为默认主来源。

最近重点内容

广告业务知识 归因口径、素材策略、案例和业务规则正在持续补充。
数据分析知识 数据字典、指标说明、报表说明和埋点文档正在持续沉淀。
项目过程知识 会议纪要、复盘记录和项目资料优先进入项目和部门知识流。

首页设计原则

  • 不做强营销，不抢用户注意力。
  • 信息量要够，但以“下一步怎么用”为主。
  • 首页更像平台门厅，而不是产品宣传页。
  • 当前版本不突出个人知识库宣传。
后面如果要更像企业平台，还可以继续补公告区、常用入口区和最近访问区。
