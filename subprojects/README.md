# 广告与数据相关项目

## 项目列表

### 广告素材情报 (ad-intelligence)
广告素材采集与分析系统
- 采集各大平台广告素材
- AI 分析（OCR、标签、文案）
- 趋势分析与竞品监控

### 广告自动化 (ad/广告自动化)
广告自动化流程与 Agent 系统
- 广告归因分析
- 用户行为追踪
- 数据链路可观测
- 广告 Agent 设计

### 连弩-AI测试平台 (chokonu)
面向 AI 应用测试、评测、自动化回归与治理闭环的平台型子项目
- 评测集与样本资产
- 评估器与规则执行
- 自动化测试与回归
- 报告、异常 Trace 与治理闭环

### AI埋点验收 (tracking-acceptance)
面向买点 / 埋点验收场景的一期 AI 工作流项目
- 文档基线与规则生成
- 验收执行与结果沉淀
- 协作提效与 AI-friendly 底座建设

### 开服系统 (server)
游戏服务器开服自动化系统
- 服务器实例自动化部署
- 开服流程自动化
- 服务器状态监控
- 自动扩缩容
