## 🗺️ 服务演进路线图

### Q2 产品规划

- 完成数据AI平台底座（MCP + CLI + Trace + 安全）
- 统一交互入口（小闪Chat），打通Agent调用能力
- 跑通3个核心业务闭环（联调 / 验收 / 诊断）
- AI原生工作流提效提质可复用

### Q3 服务规划

- 构建三大AI中台：知识中台、测试中台（TPaaS）、Agent中台（AaaS）
- 建立AI Service Layer，实现能力服务化与统一编排
- 构建AI财务数据分析与财务Agent
- AIOps全链路自动化：根因分析 + 半自动自愈服务

### Q4 服务规划

- 模型中台升级为MaaS：多模型管理 + 自动微调 + 成本优化
- 构建Agent Marketplace（Agent市场）
- 建立企业级AI服务目录（Service Catalog）
- AIOps升级为全自动闭环
