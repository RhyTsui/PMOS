# 五、开发说明

## 1、三方职责

    ┌─────────────────────────────────────────────┐
    │           Chat客户端 (MVP)                   │
    │  • 用户交互：对话、会话、结果展示             │
    │  • 自动集成：IM登录、统一大模型/MCP           │
    └─────────────────────────────────────────────┘
                        ↓
    ┌─────────────────────────────────────────────┐
    │           私有大模型 (LLM)                   │
    │  • 意图识别、工具选择、思维链、结果解释       │
    └─────────────────────────────────────────────┘
                        ↓
    ┌─────────────────────────────────────────────┐
    │           MCP服务                            │
    │  • 工具发现、SQL生成、数据查询、权限校验      │
    └─────────────────────────────────────────────┘

---

## 2、核心流程

### IM自动登录

    IM入口 → 携带IM Token → 后端验证 → 返回JWT+历史会话 → 渲染界面

### 对话流程

    用户输入 → SSE连接 → 大模型流式输出(思维链+内容+工具调用) 
           → MCP查询 → 返回结果 → 展示宽表
