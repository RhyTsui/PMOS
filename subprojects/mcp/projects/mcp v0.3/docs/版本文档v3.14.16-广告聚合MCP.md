## 一、背景

### 1.1 业务背景

广告聚合 MCP 已在运行，本次迭代基于以下正式输入合并收口：

- `3.14.16-广告聚合MCP  产品需求文档（PRD）.md`
- `需求分析文档v1.0-广告业务.md`
- `需求分析文档v1.1-广告业务.md`
- `需求Q2-广州二部.xlsx`
- `04-16 内部会议. 广告数据需求与系统化解决方案.docx`
- `纪要_04-16 内部会议. 广告数据需求与系统化解决方案.docx`
- 用户补充的原型截图与高保真原型

本轮不是新开版本，而是将现有版本统一更新为 `v3.14.16`，并按模板输出一份可直接用于评审、开发评估和汇报的版本文档。

### 1.2 现存问题

| 问题 | 具体表现 | 影响 |
| --- | --- | --- |
| 版本输入分散 | 原 PRD、分析文档、会议纪要、原型截图、映射说明分散 | 同一需求存在多套说法 |
| 筛选逻辑不统一 | 优化师、账户多选、未知值、旧智投筛选项没有统一口径 | 页面实现和工具参数难对齐 |
| 工具边界不清 | 新增报表页和智投工具只写了方向，没有标准版本说明 | 难以评估开发工作量 |
| 新增指标缺少公式 | ROI、素材、活动中的新指标只有名字，没有计算说明 | 后端实现和验收风险高 |

### 1.3 本版方案

本版按模板输出统一版本文档，固定以下口径：

1. 有效需求默认纳入本版。
2. 工期不清的放入“待开发评估工期”。
3. 交互未收敛的放入“需补原型”。
4. 页面、工具、筛选、指标、返回约束以本版文档为统一说明。

## 五、开发说明

### 5.1 通用参数定义

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `startDate` | `string` | 是 | 查询开始日期，格式 `YYYY-MM-DD` |
| `endDate` | `string` | 是 | 查询结束日期，格式 `YYYY-MM-DD` |
| `media` | `string[]` | 否 | 媒体，可多选 |
| `appType` | `string[]` | 否 | 应用类型，可多选 |
| `teamIds` | `string[]` | 否 | 团队，可多选 |
| `optimizerIds` | `string[]` | 否 | 优化师筛选，仅指定页面支持 |
| `accountIds` | `string[]` | 否 | 账户，多选不超过 10 个 |
| `campaignIds` | `string[]` | 否 | 广告组，多选不超过 10 个 |
| `adIds` | `string[]` | 否 | 广告，多选不超过 10 个 |
| `activityIds` | `string[]` | 否 | 活动，多选不超过 10 个 |
| `packageIds` | `string[]` | 否 | 包体/分包，多选不超过 10 个 |
| `groupBy` | `string[]` | 否 | 细分维度 |
| `metrics` | `string[]` | 否 | 返回指标列表 |
| `topN` | `number` | 否 | 指标 TopN，仅支持 `< 10` |
| `withSummary` | `boolean` | 否 | 是否返回汇总行，默认 `true` |
| `pageNo` | `number` | 否 | 页码，默认 `1` |
| `pageSize` | `number` | 否 | 每页条数，默认 `10`，最大 `10` |

### 5.2 通用约束

1. 账户、广告组、广告、活动、包体等多选项最多支持选择 10 个。
2. 账户/广告/素材等明细报表最多返回 10 条。
3. 细分结果最多返回 10 条。
4. 支持返回汇总行。
5. 支持指标 TopN 筛选，且 `topN < 10`。
6. 新增智投 MCP 工具默认返回第一页 10 条内容。
7. 广告和素材聚合分析默认优先从账户汇总，其次从广告组汇总。

### 5.3 优化师筛选支持范围

- 支持：日周月报、广告素材、ROI、LTV、留存、活动、包体、直播、关键词
- 不支持：实时数据、回流数据、回流LTV、回流ROI
- 匹配不上按“未知”处理，且“未知”不进入筛选结果列表

### 5.4 友好提示

| 场景 | 提示语 |
| --- | --- |
| 多选超过 10 个 | 当前最多支持选择 10 个筛选项，请缩小范围后重试。 |
| 明细返回超过 10 条 | 当前仅展示前 10 条明细，请增加筛选条件或改用汇总视图。 |
| 细分结果超过 10 条 | 当前细分结果最多返回 10 条，请增加筛选条件或使用 TopN。 |
| TopN 大于等于 10 | TopN 仅支持小于 10 的值，请调整后重试。 |
| 新增智投工具首次打开 | 当前默认返回第一页 10 条内容，可继续翻页或增加筛选条件。 |
| 筛选项出现未知值 | 当前筛选结果默认隐藏“未知”项，仅展示已识别的筛选值。 |

### 5.5 新增指标计算说明

| 指标 | 计算说明 | 异常处理 |
| --- | --- | --- |
| `7日付费次数` | 统计周期内满足当前筛选条件的用户，在注册后 7 日内产生的付费总次数；汇总行按明细求和 | 无分母问题，按数据源直接汇总 |
| `7日人均付费次数` | `7日付费次数 ÷ 7日付费人数` | `7日付费人数 = 0` 时返回 `-` |
| `7日付费成本` | `折后消耗 ÷ 7日付费次数` | `7日付费次数 = 0` 时返回 `-` |
| `ltv3/ltv1` | `LTV3 ÷ LTV1` | `LTV1 = 0` 时返回 `-` |
| `ltv7/ltv1` | `LTV7 ÷ LTV1` | `LTV1 = 0` 时返回 `-` |
| `ltv7/ltv3` | `LTV7 ÷ LTV3` | `LTV3 = 0` 时返回 `-` |
