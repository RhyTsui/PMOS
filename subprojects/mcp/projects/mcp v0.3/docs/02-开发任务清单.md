## 执行说明

本清单与 `prd.md` 保持一致，按以下口径推进：

## 五、当前跟进重点

1. 优先推进标签、基础查询与核心报表能力，保证本版可交付主链路。
2. 当前新增里优先级最高的是 `7日付费相关指标` 与 `素材报表筛选项`。
3. `有效用户相关指标` 保留在版本内，但不作为这轮主收口项。
4. 多维分析相关需求统一收口到开发评估，不再以聚合分析引擎 MCP 形式拆任务。
5. 广州二部需求与需求池新增需求均视为本版范围，不能挂起到后续版本。
6. 原型需求只作为前置交付物，不影响其进入 `v3.14.16` 的版本归属。
7. 会议纪要中已确认的共性指标需求，优先直接转开发任务，不再停留在口径讨论阶段。
8. 会议纪要中需跨团队协同的事项，统一进入评估任务跟踪，不直接写成已支持。
9. 三大媒体字段映射作为本版基础数据治理任务处理，不等同于后续实现时再临时口头约定。
10. 原型截图中已出现的交互，不视为“默认合理”；仍需收敛成正式原型需求和执行任务。
