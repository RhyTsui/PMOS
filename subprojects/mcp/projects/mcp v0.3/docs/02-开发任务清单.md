# 广告聚合 MCP v1.0 - 开发任务清单

## 任务拆解

### Phase 1: 报表 MCP 工具 (2 天)

#### T1.1 get_ad_hour_report - 广告小时报表
- **输入**: 日期范围、广告 ID、优化师 ID
- **输出**: 小时级消耗、展示、点击、转化数据
- **依赖**: 广告 BI 小时表

#### T1.2 get_ad_day_report - 广告日/周/月报表
- **输入**: 日期范围、聚合维度（日/周/月）、筛选条件
- **输出**: 聚合后的核心指标
- **依赖**: 广告 BI 日表 + 向上聚合逻辑

#### T1.3 get_ad_roi_report - 广告 ROI 报表
- **输入**: 日期范围、项目 ID、回流数据源
- **输出**: ROI、LTV、回收周期
- **依赖**: 广告消耗数据 + 回流数据

#### T1.4 get_ad_retention_report - 广告留存报表
- **输入**: 日期范围、留存类型（次日/7 日/30 日）
- **输出**: 留存率、留存用户数
- **依赖**: 用户行为数据

---

### Phase 2: 标签 MCP 工具 (1 天)

#### T2.1 list_team - 广告团队/优化师列表
- **输入**: 团队 ID（可选）
- **输出**: 优化师列表及基本信息

#### T2.2 list_apps - 项目/应用列表
- **输入**: 无
- **输出**: 项目/应用列表

#### T2.3 list_media - 广告媒体列表
- **输入**: 媒体类型（可选）
- **输出**: 媒体渠道列表

---

### Phase 3: 推送 MCP 工具 (1 天)

#### T3.1 send_report_push - 报表推送
- **输入**: 报表类型、推送范围、推送时间、接收人
- **输出**: 推送结果
- **依赖**: 推送渠道配置

---

### Phase 4: 联调测试 (1 天)

#### T4.1 端到端测试
- 报表工具链调用
- 标签筛选组合测试
- 推送全流程验证

#### T4.2 性能测试
- 大数据量聚合查询
- 并发调用测试
