# 季度 OKR 知识库

此文件夹用于存储**产品负责人的季度 OKR**（Objectives and Key Results），是 PMAIOS 制定产品规划和需求决策的核心依据。

## 定位

- **录入**：产品负责人的季度 OKR，包含各季度产品达成目标计划
- **用途**：PMAIOS 所有产品规划、需求设计、项目实施都**紧紧围绕 OKR 来制定和执行**
- **核心**：确保产品方向与公司战略目标一致

## 支持格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| ✅ Excel | `.xlsx`, `.xls` | **推荐** - OKR 表格专用格式 |
| ✅ Markdown | `.md` | 文本描述格式 |
| ✅ DOCX | `.docx` | Word 文档格式 |

## 使用说明

1. **文件命名格式**：`YYYY-Q 季度-OKR.xlsx` 或 `YYYY-Q 季度-OKR.md` 或 `YYYY-Q 季度-OKR.docx`
2. **制定时机**：每季度初制定，季度中可微调，季度末复盘
3. **Excel 格式**：PMAIOS 自动解析 OKR 表格结构

## PMAIOS 集成

| 功能 | 说明 |
|------|------|
| **Excel 解析** | 自动读取 OKR 表格结构 |
| **OKR 对齐检查** | 所有产品规划必须关联到具体 OKR |
| **需求优先级** | 基于 OKR 权重自动计算需求优先级 |
| **进度追踪** | 周报、项目进展自动关联 OKR 完成度 |
| **季度复盘** | AI 辅助生成季度 OKR 复盘报告 |

## 与周报的关系

```
季度 OKR (方向)
    ↓
产品规划 (路径)
    ↓
周工作 (执行)
    ↓
周报 (反馈)
    ↓
OKR 进度更新 (闭环)
```
