# 会议纪要知识库

此文件夹用于存储公司各项目的会议纪要，供 PMAIOS 和各个子项目共享读取。

## 使用说明

1. 会议纪要文件命名格式：`YYYY-MM-DD-项目名称-会议主题.md`
2. 支持格式：Markdown (.md)、DOCX (.docx)、Excel (.xlsx)
3. 所有会议纪要将作为 PMAIOS 的上下文知识，用于系统设计和需求决策

## 目录结构

```
meeting-notes/
├── README.md                    # 本文件
├── 2026/
│   ├── 04/
│   │   ├── 2026-04-10-PMAIOS-v0.4 规划会议.md
│   │   └── 2026-04-13-XX 项目需求评审.md
│   └── ...
└── ...
```

## 会议纪要模板

```markdown
# 会议纪要

## 基本信息
- 日期：YYYY-MM-DD

- 参会人员：...
- 会议主题：...

## 讨论内容
...

## 决策事项
...

## 待办事项
- [ ] ...
```

## PMAIOS 集成

PMAIOS 通过以下方式读取会议纪要：

1. **ChromaDB 向量检索** - 自动索引所有会议纪要
2. **上下文增强** - 在生成 PRD 时参考历史决策
3. **需求追溯** - 关联需求与原始会议讨论
