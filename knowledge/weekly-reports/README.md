# 周报知识库

此文件夹用于存储**产品负责人的周汇报文档**，供 PMAIOS 读取并辅助生成后续周报。

## 定位

- **录入**：产品负责人每周的工作汇报（手动录入或 AI 生成）
- **用途**：PMAIOS 读取历史周报，辅助生成下周周报、追踪项目进展
- **核心**：围绕产品 OKR 展开，确保每周工作与季度目标对齐

## 使用说明

## 使用说明

1. **文件命名格式**：`YYYY-MM-DD-周报.md` 或 `YYYY-W 周数 - 周报.md`
2. **支持格式**：Markdown (.md)、DOCX (.docx)、Excel (.xlsx)
3. **AI 辅助生成**：PMAIOS 可基于本周工作记录自动生成周报草稿

## PMAIOS 集成

| 功能 | 说明 |
|------|------|
| **历史读取** | ChromaDB 向量检索，读取历史周报 |
| **AI 生成草稿** | 基于本周工作记录自动生成周报 |
| **OKR 对齐检查** | 确保周报内容围绕季度 OKR |
| **进展追踪** | 关联多周数据，生成趋势分析 |
