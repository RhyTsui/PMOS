```

→ 步骤5: 进入账户详情 → 步骤6: 进入事件管理 → 步骤7: 处理声明页面 
→ 步骤8: 关闭引导 → 步骤9: 选择资产联调 → 步骤10: 选择监测组 
→ 步骤11: 生成二维码 → 步骤12: 发送飞书
```

- ✓ 弹框边缘有阴影效果

| **关闭后弹框仍存在** | 关闭操作后仍检测到弹框元素 | 再次尝试关闭，最多3次；仍失败则继续 |

#### 【智能检测函数】
```python
async def close_modal_smartly(page):
    """智能检测并关闭弹框（完整版）"""
    
    logger.info("[弹框处理] 开始检测页面弹框...")
    
    # 1. 等待页面稳定
    await page.wait_for_load_state('networkidle')
    await page.wait_for_timeout(3000)  # 额外等待3秒
    
    # 2. 检测弹框是否存在
    modal_indicators = [
        "div[class*='modal'], div[class*='dialog'], div[class*='popup']",
        "text=旧版即将下线",
        "text=升级",
        "text=下线安排",
        ".modal-backdrop, .mask, .overlay"
    ]
    
    modal_found = False
    for indicator in modal_indicators:
        try:
            if await page.is_visible(indicator, timeout=2000):
                modal_found = True
                logger.info(f"[弹框处理] ✓ 检测到弹框特征: {indicator}")
                break
        except:
            continue
    
    if not modal_found:
        logger.info("[弹框处理] 未检测到弹框，继续下一步")
        return False
    
    # 3. 截图保存
    await page.screenshot(path="debug_modal_detected.png")
    
    # 4. 记录弹框内容
    try:
        modal_text = await page.locator('.modal, .dialog').first.text_content()
        logger.info(f"[弹框处理] 弹框内容: {modal_text[:100]}...")
    except:
        pass
    
    # 5. 尝试关闭（多种方式）
    close_selectors = [
        "button.close",
        "button[aria-label='关闭']",
        "button[aria-label='Close']",
        ".modal-close",
        ".dialog-close",
        "[class*='close-btn']",
        "//div[contains(@class, 'modal')]//button[contains(@class, 'close')]",
        "//div[contains(@class, 'dialog')]//button[@aria-label='关闭']",
        "svg.close",
        "[class*='icon-close']",
    ]
    
    for idx, selector in enumerate(close_selectors):
        try:
            if await page.is_visible(selector, timeout=1000):
                await page.click(selector)
                logger.info(f"[弹框处理] ✅ 成功关闭弹框，使用定位器 #{idx+1}: {selector}")
                await page.wait_for_timeout(500)
                await page.screenshot(path="debug_modal_after_close.png")
                return True
        except Exception as e:
            logger.debug(f"[弹框处理] 定位器 #{idx+1} 失败: {selector}")
            continue
    
    # 6. 尝试ESC键
    try:
        await page.keyboard.press('Escape')
        logger.info("[弹框处理] 尝试按ESC键关闭")
        await page.wait_for_timeout(500)
        return True
    except:
        pass
    
    # 7. 尝试点击遮罩层
    try:
        await page.click('.modal-backdrop, .mask, .overlay', position={'x': 10, 'y': 10})
        logger.info("[弹框处理] 尝试点击遮罩层关闭")
        return True
    except:
        pass
    
    # 8. 所有方式均失败
    logger.warning("[弹框处理] ⚠️ 无法关闭弹框，记录并继续执行")
    await page.screenshot(path="debug_modal_failed.png")
    return False
```

#### 【成功判断】
- ✓ 弹框元素从页面消失
- ✓ 遮罩层消失
- ✓ 页面恢复正常交互状态

| 异常类型 | 检测条件 | 处理策略 |
|---------|---------|---------|
| **文件不存在** | 二维码文件路径无效 | 记录错误，跳过发送 |
| **Webhook无效** | 飞书API返回错误 | 检查Webhook URL是否正确 |
| **网络异常** | 请求超时或失败 | 重试发送，最多3次 |
| **权限不足** | API返回权限错误 | 检查机器人权限配置 |
