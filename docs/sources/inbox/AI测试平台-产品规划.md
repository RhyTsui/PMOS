1、定位

AI 自动化质量控制平台

一句话：用 AI 驱动需求理解、用例生成、自动执行、结果判定、问题归因和上线门禁，覆盖传统系统、数据链路、AI 应用、Agent、MCP、知识库、Workflow 的统一质量控制平台。





2、框架

主框架：需求 → 用例 → 执行 → 结果 → 报告 → 分析 → 复核 → 门禁



能力域：UI / API / 数据链路 / 真机 / RAG / Agent / MCP / Workflow / Skill



底座：统一测试资产 + 执行编排 + Trace观测 + AI Judge + 质量门禁
