## 关键提醒

### 1. attr_click表没有device_id

这是最重要的业务约束：
- 媒体点击表是原始点击数据，没有device_id字段
- 必须通过idfa/oaid/android_id查询点击表
- 诊断工具需要先查设备表获取idfa，再关联点击表

### 2. 时序约束

正常链路必须满足：
```
点击时间 < 激活时间 < 注册时间 < 付费时间
```

### 3. ID关联

```
点击表(idfa/oaid) → 设备表(idfa, device_id) → 其他表(device_id)
```
