07 AI Coding Prompt

重要：所有图标必须使用 SVG，禁止使用 Emoji。使用图标库如 Ant Design Icons (@ant-design/icons)、Lucide (lucide-react) 或 Heroicons。

🤖 System Prompt（复制到 AI Coding 工具）
# 魔方问数 AI助手界面生成约束 # 版本: 2.0 | 日期: 2026-05-11 ## 1. 图标规范（重要！） ### 禁止使用 Emoji，必须使用 SVG 图标 使用图标库（推荐 Ant Design Icons 或 Lucide）： - Logo: 自定义 SVG，28×28px，蓝色背景，白色图标 - 消息图标: <MessageOutlined /> 或 MessageCircle - 图表图标: <BarChartOutlined /> 或 BarChart3 - 通知: <BellOutlined /> 或 Bell - 帮助: <QuestionCircleOutlined /> 或 HelpCircle - 设置: <SettingOutlined /> 或 Settings - 技能: <ThunderboltOutlined /> 或 Zap - 发送: <SendOutlined /> 或 Send - 加号: <PlusOutlined /> 或 Plus - 下拉: <DownOutlined /> 或 ChevronDown ## 2. 色彩 --brand: #4A7CFF --sidebar-bg: #F8F9FB // 浅色！ --text-primary: #1A1A2E ## 3. 组件关键样式 ### Logo 图标: 28×28px, background:#4A7CFF, border-radius:6px 文字: "魔方问数", 17px, font-weight:700 ### Tab切换器 容器: background:#F3F4F6, border-radius:8px, padding:3px 选项: padding:7px 18px 选中: background:#fff, color:#4A7CFF, box-shadow 图标: 14px SVG（消息气泡/柱状图） ### 历史会话 图标: 14px 消息气泡 SVG 标题: 14px, 单行截断 时间: 12px, #9CA3AF, margin-left:auto ### 筛选标签 运营: 文字 + 下拉箭头（8px） 分析项目: 加号图标 + 文字 指间山海: 粉色圆点（#EC4899, 8px）+ 文字 + 下拉箭头 ### 发送按钮 胶囊形: border-radius:9999px 图标: 纸飞机 SVG（白色） 文字: "发送", 13px ## 4. 布局 侧边栏: 260px, background:#F8F9FB（浅色！） 主内容: flex:1, background:#fff ## 5. 输出要求 1. 使用 CSS 变量（--brand 等） 2. 所有图标用 SVG，禁止 Emoji 3. 侧边栏必须是浅色 #F8F9FB 4. 代码注释清晰
