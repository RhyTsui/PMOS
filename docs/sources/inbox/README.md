# Input Inbox

这是 `PMAIOS` 的唯一人工输入投递目录。

以后你如果有新的原始材料，统一先放这里，我再负责读取、提炼、搬运到正式位置。

适合直接丢进来的内容：

- 会议纪要
- 版本分析
- 需求分析
- 架构草稿
- 外部摘录
- 任何尚未定稿、但希望我帮你整理的输入材料

处理约定：

- 这里是 **输入源投递口**，不是最终归档位置
- 我读取后，会根据内容搬运到 `knowledge/`、`docs/sources/`、`docs/decisions/` 或其他正式目录
- 若内容带有明确规则，我会同步提炼进全局规则
- 若内容带有明确需求，我会考虑沉淀进需求池

当前唯一入口路径：

- `docs/sources/inbox/`

如果你不确定材料属于平台、某个项目、还是多个项目联动：

- 不要自己先分目录
- 直接先丢到这里
- 再告诉我“读取 inbox，提炼并搬运”
