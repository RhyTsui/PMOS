# Sources

此目录用于存放 **PMAIOS 的输入源文档**，不是最终决策目录。

这里的文档可以是：

- 人工分析
- 版本推演
- 候选方案
- 会议整理后的结构化输入
- 尚未定稿的架构/需求/实施思考

规则：

- 可并存多个版本，不要求“唯一正确”
- 允许相互冲突
- 允许是草稿
- 默认只作为 AI 与人工决策的输入材料
- 不直接作为执行真源

真正生效的系统决策请看：

- `docs/decisions/`

真正驱动执行的基线请看：

- `workflows/`
- `docs/tasks/`
- 少量已确认的 `docs/implementation/` 文档
