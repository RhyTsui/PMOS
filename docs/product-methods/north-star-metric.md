## 1. 解决什么问题

当团队一直在做事，但说不清产品真正成功的主指标时，`North Star Metric` 用来统一价值判断口径。

## 2. 适用场景

- 需要建立产品长期价值指标
- 团队在优化局部指标，但缺总方向
- 想把产品工作和价值闭环连接起来

## 3. 不适用场景

- 还没定义清楚产品目标用户和核心任务
- 当前只是局部需求优先级比较

## 4. 输入

- 产品目标
- 用户核心行为
- 价值产生方式
- 现有指标体系

## 5. 输出

- north star metric
- metric rationale
- guardrail metrics
- measurement risks

## 6. 产物挂点

- 产品价值指标定义
- 版本目标说明
- 成功标准章节
