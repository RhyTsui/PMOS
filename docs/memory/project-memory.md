# Project Memory

## Long-term facts

- Project root is `E:\\AI\\ai-os`
- Stack baseline is Node.js + TypeScript + React + Vite
- OpenSpec and Superpowers are external integrations, not in-repo reimplementations
- AI OS workflow is file-driven and phase-based

## Working agreements

- Artifacts must be written to files
- Review committee is a required gate
- Provider credentials must come from environment variables
- 默认使用中文沟通、中文展示与中文规划表达；仅保留少量必要英文术语
- 路径规则、作用域规则、目录规则等 OS 层定义优先用中文编写
- 记忆能力按底层通用能力处理，面向 PMAIOS 与其子项目共同服务
- `docs/sources/inbox/` 是唯一人工输入投递入口
- 对话中形成的全局规则应沉淀到 `docs/memory/global-rules.md`
- 对话中明确提出的需求应优先进入 `docs/memory/requirements/` 作为 draft requirement
