# PMOS v1.0 Product Version Program

## Goal

交付一个独立的 `PMOS product repo`，让它可以被部署、配置、分享和理解，而不依赖原始内部仓里的隐藏上下文。

## 必须成立的结果

1. 独立产品仓边界成立
2. 配置以 `sample/example` 形式公开
3. 部署文档和首跑文档齐全
4. `GitHub + Notion + cloud-mirror` 云端知识分层成立
5. 首版安装、构建、启动可复现

## 当前已完成

1. 产品仓边界、保留/排除清单已落真源
2. README、部署说明、发布清单已落真源
3. 独立 `product-repo` 已完成首版代码骨架组装
4. 独立目录内 `npm install / lint / build / start` 已跑通
5. 产品仓与主仓的最新 `v1.0` 真源同步机制已进入持续补丁阶段

## 当前策略

`先发布首版可用产品仓，再对未闭环项持续打补丁。`

这意味着：

1. 已完成的独立产品仓能力不再回退成“只是规划”
2. 尚未完全闭环的项进入后续补丁，不阻塞首版产品仓主线
3. 主仓里继续演进的 `v1.0` 真源，需要持续同步回产品仓
