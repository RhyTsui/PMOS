- version: v0.1
- date: 2026-05-09
- status: active
- owner: product office

## 1. 目标

首版 `PMOS product repo` 必须做到：

1. 可被外部读懂
2. 可按文档部署
3. 可作为 ChatGPT 的 GitHub 主知识源
4. 不混入业务子项目主体
