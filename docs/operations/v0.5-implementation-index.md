# PMAIOS v0.5 实施索引

- version: v0.6
- date: 2026-04-22
- status: active draft
- purpose: 汇总当前 `v0.5` 已经收敛出来的实施草案、模板、协议、图板和深研入口，避免能力散落

## Scope

这份索引面向 `v0.5` 的实施层。

它不是：

- 版本愿景海报
- 公司演示主图
- 单纯路线图说明

它是把当前 `v0.5` 的能力块转成可接入、可治理、可持续推进的统一入口。
